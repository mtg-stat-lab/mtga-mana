Methodology
Single-Deck Approach

All spells, lands, and mana-producing spells appear in one deck JSON. For example:

{
  ">B": 8,          // 8 basic lands that produce B
  "BW": 2,            // 2 spells costing B+W
  "3*>WUBRG": 1    // 1 spell costing 3 colorless that can then tap for W/U/B/R/G
}

The simulator draws cards turn by turn, tracks how many spells are "dead," and once a mana-producing spell is deemed castable at turn X, it counts as a permanent mana source for all subsequent turns.
