Set
Select the card set to use in the simulation. Different sets may have different card data and rules.
Play or Draw
Choose whether you are on the play or on the draw. This affects the simulation's draw order and available mana.
Deck List
Paste your deck list here. It should include a "Deck" section and optionally a "Sideboard" section (which will be ignored).
Deck Size
Specify the total number of cards in the deck, including filler cards. This determines how many cards are in your simulated deck.
Initial Hand Size
Enter the number of cards drawn in your opening hand.
Number of Draw Steps
This is the number of turns (or draw steps) to simulate beyond the initial hand.
Number of Simulations
Set how many times the simulation will run. More simulations provide smoother statistics but take longer.
Random Seed
Enter a number to seed the random number generator. This makes simulation runs reproducible.
